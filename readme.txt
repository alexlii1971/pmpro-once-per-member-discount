== Description ==
A plugin for Paid Membership Pro which adds the ability to restrict a discount code to be used only once per member account.

The plugin is based on the code provided by PMPro: http://www.paidmembershipspro.com/restrict-disco-time-per-user/

A check box is added to the wordpress PMPro discounts settings page to activate the restriction per code.

== Installation ==
1. Upload the 'pmpro-once-per-member-discount' folder to the '/wp-content/plugins/' directory of your site.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
